# 3D portfolio

An interactive React portfolio experiment built with Vite, Three.js, Tailwind, and motion focused UI.

## Product problem

A developer portfolio has one job: make the strongest work easy to remember. This project explores a more visual version of that surface with 3D presentation, animated sections, project cards, and a modern frontend stack.

## What it shows

| Area | Evidence |
| --- | --- |
| Modern frontend tooling | Vite, React, Tailwind, Three.js, React Three Fiber, Drei, and Framer Motion dependencies. |
| Presentation design | Hero, about, experience, tech, work, feedback, and contact sections. |
| Project storytelling | Curated work is framed around business workflows, APIs, dashboards, and full stack execution. |
| Recruiter surface | The site gives a second visual path into the same job hunting narrative. |

## Stack

React, Vite, Three.js, React Three Fiber, Drei, Tailwind CSS, Framer Motion, EmailJS.

## Run locally

```bash
npm install
npm run dev
```

Build for production.

```bash
npm run build
```
