# 3D Developer Portfolio

An interactive portfolio experiment built with React, Vite, Three.js, React Three Fiber, Tailwind CSS, and Framer Motion.

## Product story

This project shows Gokhan's interest in polished frontend experiences beyond standard pages. It uses motion and 3D presentation to make a developer portfolio feel more memorable while still highlighting projects, skills, and contact flow.

## What it demonstrates

| Area | Evidence |
| --- | --- |
| Modern frontend tooling | Vite, React 18, module based source structure, and Tailwind. |
| 3D interaction | React Three Fiber and Drei render 3D scenes and animated visuals. |
| Motion design | Framer Motion supports animated sections and page transitions. |
| Portfolio UX | Project cards, experience sections, and contact components create a full presentation flow. |

## Stack

React, Vite, Three.js, React Three Fiber, Drei, Tailwind CSS, Framer Motion, EmailJS.

## Run locally

```bash
npm install
npm run dev
```

Create a production build.

```bash
npm run build
```
